# Responsive Layout and Cross-Browser Testing Checklist

## Browsers to Test

- [ ] Chrome (latest)
- [ ] Firefox (latest)
- [ ] Safari (latest)
- [ ] Edge (latest)
- [ ] Mobile Safari (iOS)
- [ ] Chrome for Android

## Layout Breakpoints

- [ ] Desktop (≥1200px)
- [ ] Tablet (768px–1199px)
- [ ] Mobile (≤767px)

## UI Elements

- [ ] Canvas fills viewport at all breakpoints
- [ ] NavBar is always visible and does not overlap content
- [ ] SidePanel opens/collapses correctly at all sizes
- [ ] Hamburger button is accessible and visible at all sizes
- [ ] No horizontal scrollbars appear at any breakpoint

## Interactions

- [ ] Panning/zooming works on all browsers/devices
- [ ] SidePanel can be toggled with mouse, touch, and keyboard
- [ ] NavBar actions are clickable/tappable

## Accessibility

- [ ] All interactive elements are focusable and operable by keyboard
- [ ] ARIA labels are present and correct

## Notes

- Record any browser-specific issues and file bugs as needed.
